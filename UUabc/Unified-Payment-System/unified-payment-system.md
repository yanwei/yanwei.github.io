# 统一外教薪资系统

<link rel="stylesheet" href="https://yanwei.github.io/auto-number-title.css" />

<p align='right'>v1.0, Last Update: 2018.12.01</p>

## 目标

* 统一1.0、1.5、2.0等各个不同系统的外教薪资计算
* 简化处理和计算逻辑，降低人工成本和差错率
* 适应未来可能的规则调整，做到“改变薪资规则无需发版，运营后台配置即可”
* 更强大和灵活的报表系统
